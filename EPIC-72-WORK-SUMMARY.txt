ARCHITECTURE
───────────────────────────────────────────────────────────────────────────────

System Architecture:

  Ollama (external)
      │ AI decisions
      ▼
  Chess Arena (arena.js)
      │ Match management
      ├─→ RealChessGame (play moves)
      │   └─→ Ollama (queries for moves)
      │       └─→ emitMovePlayed()
      │
      └─→ WebSocket Server (port 9000)
          └─→ Broadcasts to all clients
              └─→ Browser WebSocket (localhost:5173)
                  └─→ React hooks update state
                      └─→ ChessSpectator renders

Data Flow for Each Move:

  1. RealChessGame.executeMove()
  2. wsServer.emitMovePlayed(moveData, player, latency)
  3. WebSocket broadcasts: {type: 'MovePlayed', fen, move, moveNumber, ...}
  4. Browser receives via WebSocket.onmessage()
  5. useWebSocket.ts updates gameState.fen
  6. ChessSpectator.tsx rerenders with new FEN
  7. Chessboard displays new position (<50ms latency)
