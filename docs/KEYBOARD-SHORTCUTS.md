# AI Commander Broadcast Keyboard Shortcuts

Quick reference for all keyboard shortcuts available during AI Commander broadcasting.

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `H` | Toggle HUD | Show/hide game state HUD (resources, military, tech, population) |
| `Ctrl+Shift+S` | Toggle Stream Mode | Hide developer tools, show spectator-only view (ideal for broadcasting) |
| `M` | Toggle Minimap | Show/hide minimap overlay |
| `Escape` | Close Dialogs | Close any open dialogs or menus |

## Playback Control

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Space` | Play/Pause | Pause/resume live match or replay |
| `→` | Seek Forward (5s) | Jump forward 5 seconds in replay |
| `←` | Seek Backward (5s) | Jump backward 5 seconds in replay |
| `Shift+→` | Seek Forward (30s) | Jump forward 30 seconds in replay |
| `Shift+←` | Seek Backward (30s) | Jump backward 30 seconds in replay |
| `Home` | Jump to Start | Go to beginning of match |
| `End` | Jump to End | Go to end of match |

## Camera Control

| Shortcut | Action | Description |
|----------|--------|-------------|
| `C` | Toggle Cinematic Mode | Enable/disable automatic cinematic camera |
| `+` | Increase Camera FOV | Zoom out (wider view) |
| `-` | Decrease Camera FOV | Zoom in (narrow view) |
| `R` | Reset Camera | Reset to default position |

## Replay & Slow Motion

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+R` | Trigger Instant Replay | Replay last 30 seconds of gameplay |
| `[` | Decrease Speed | Slow down playback (0.25x steps) |
| `]` | Increase Speed | Speed up playback (0.25x steps) |
| `1` | 0.25x Speed | Quarter speed (cinematic slow-motion) |
| `2` | 0.5x Speed | Half speed |
| `3` | 1.0x Speed | Normal speed (live) |
| `4` | 1.5x Speed | 1.5x speed (highlight reels) |
| `5` | 2.0x Speed | Double speed (fast-forward) |

## Display Options

| Shortcut | Action | Description |
|----------|--------|-------------|
| `T` | Toggle Timeline | Show/hide decision timeline (left side) |
| `O` | Toggle Objective Tracker | Show/hide AI strategy evolution (right side) |
| `E` | Toggle Event Annotations | Show/hide major events overlay |
| `A` | Toggle AI Status | Show/hide AI player info (latency, confidence, model) |

## Commentary & Narration

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+C` | Toggle Commentary | Enable/disable live esports commentary audio |
| `Ctrl+N` | Toggle Narration Subtitles | Show/hide on-screen commentary text |
| `L` | Toggle Commentary Log | Show/hide historical commentary entries |

## View Presets

| Shortcut | Action | Description |
|----------|--------|-------------|
| `V` then `1` | Minimal View | Hide all UI except game footage |
| `V` then `2` | Broadcast View | Show HUD + AI status + overlay (default streaming view) |
| `V` then `3` | Analysis View | Show all telemetry (HUD, timeline, objective, events, commentary) |
| `V` then `4` | Narrative View | Show story-focused view (match narrative, key moments) |

## Match Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `N` | Next Match | Load next match from queue |
| `P` | Previous Match | Load previous match from history |
| `Ctrl+L` | Match List | Show available matches to load |
| `Ctrl+Shift+L` | Live Match | Jump to currently broadcasting match |

## Settings & Configuration

| Shortcut | Action | Description |

## Tips for Broadcasters

### Essential Shortcuts for Live Broadcasting
Keep these at hand for common broadcast tasks:

1. **`Ctrl+Shift+S`** — Stream Mode (start your broadcast with this)
2. **`Space`** — Play/Pause (pause on dramatic moments for commentary)
3. **`Ctrl+R`** — Instant Replay (show key moments again)
4. **`[` / `]`** — Speed control (slow-motion for cinematic effect)
5. **`H`** — Toggle HUD (hide stats when discussing strategy)

### Suggested Key Binding by Role

**Live Commentary (Esports Caster):**
- Always keep `Ctrl+Shift+S` active for clean broadcast view
- Use `Space` to pause on dramatic moments
- Use `Ctrl+R` to replay key moments
- Use `[/]` to control pacing

**Technical Director (OBS):**
- Use `V+2` to switch to broadcast view for streaming
- Use `V+3` to switch to analysis view for educational content
- Keep `Escape` handy to close accidental dialogs
- Monitor with `Ctrl+?` reference open

**Match Analyst:**
- Use `T` to show/hide timeline for decision analysis
- Use `O` to study objective evolution
- Use `E` to review event annotations
- Use `Ctrl+C` to toggle commentary for note-taking

### Keyboard Shortcut Tips

1. **Avoid conflicts** — These shortcuts don't conflict with OBS hotkeys or Twitch chat
2. **Practice timing** — Get comfortable with key combinations for smooth broadcast
3. **Customize** — Settings → Keyboard Shortcuts to reassign keys if needed
4. **Display guide** — Press `Ctrl+?` during broadcast to remind yourself of key bindings

## Troubleshooting Shortcuts

**If a shortcut doesn't work:**

1. Make sure the AI Commander window is focused (click on it)
2. Check if Stream Mode is active (may disable some shortcuts)
3. Verify no modifiers (Ctrl, Shift, Alt) are stuck
4. Restart the browser tab: `Ctrl+Shift+R` (hard refresh)

**If you need to see current shortcuts:**

1. Press `Ctrl+?` to open the shortcuts help panel
2. Check Settings → Keyboard Shortcuts for current bindings

---

## Custom Shortcuts (Advanced)

To customize keyboard shortcuts:

1. **Open Settings** (`Ctrl+,`)
2. **Keyboard → Custom Shortcuts**
3. Click any shortcut to reassign it
4. Press your desired key combination
5. **Save** and restart browser tab

**Note:** Custom shortcuts are saved to browser localStorage and persist across sessions.
