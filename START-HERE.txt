INITIAL SETUP (One Time)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install prerequisites:
   - Node.js 18+: https://nodejs.org/
   - pnpm: npm install -g pnpm
   - Ollama: https://ollama.ai/

2. Build the project:
   cd C:\Users\boter\ai-commander
   pnpm install
   pnpm build

3. Start Ollama in a separate terminal (KEEP RUNNING):
   ollama serve

COMMAND 1: RUN A MATCH
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ai-commander match start [OPTIONS]

Basic:
   ./ai-commander.bat match start

With options:
   ./ai-commander.bat match start \
     --brain1 "Ollama" \
     --brain2 "Ollama" \
     --max-ticks 5000

What happens:
   - Game window opens
   - Two AI brains make decisions
   - Match completes (45-60 seconds)
   - Replay saved to ./replays/

Options:
   --brain1 <name>       First AI (default: Ollama)
   --brain2 <name>       Second AI (default: Ollama)
   --max-ticks <num>     Game length (default: 5000)
   --no-window           Skip game window
   --replay-dir <path>   Save location (default: ./replays)

COMMAND 2: RUN A TOURNAMENT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ai-commander tournament run [OPTIONS]

Using presets (easiest):
   ./ai-commander.bat tournament run --preset ollama-vs-ollama
   ./ai-commander.bat tournament run --preset quick-match

Manual:
   ./ai-commander.bat tournament run \
     --brains "Brain1,Brain2,Brain3" \
     --format round_robin

Available presets:
   - ollama-vs-ollama    Two local Ollama instances
   - multi-llm           Ollama + Claude + GPT
   - builtin-vs-ollama   Builtin AI vs Ollama
   - quick-match         Fast 1000-tick match
   - long-match          Extended 10000-tick match

What happens:
   - Multiple matches run with rankings
   - ELO ratings calculated
   - Replays saved to ./tournament-replays/
   - 45 seconds per match

Options:
   --preset <name>       Use preset config
   --brains <list>       Comma-separated names
   --format <fmt>        round_robin or single_elimination
   --max-ticks <num>     Ticks per match
   --name <name>         Tournament name

COMMAND 3: EXPORT RESULTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ai-commander replay export <match-id> --format [FORMAT]

To HTML (best for viewing):
   ./ai-commander.bat replay export match-001 --format html
   start match-001.html

To CSV (for spreadsheets):
   ./ai-commander.bat replay export match-001 --format csv

To JSON (raw data):
   ./ai-commander.bat replay export match-001 --format json

All formats:
   ./ai-commander.bat replay export match-001 --format json,csv,html

What happens:
   - Replay converted to requested format
   - File saved (match-001.html, match-001.csv, match-001.json)
   - <1 second
