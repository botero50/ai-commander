## Quick Start

### 1. Verify Everything Works

```bash
# Check Ollama is running
ollama list

# Check voice model exists
ls -lh packages/zeroad-adapter/voices/

# Test Piper TTS
echo "Test" | piper --model en_US-lessac-medium --data-dir packages/zeroad-adapter/voices --output-file test.wav
```

### 2. Run Your First Tournament

```bash
# Run 3 matches
npx tsx packages/zeroad-adapter/src/arena/run-arena-loop.ts --matches 3

# Or run forever
npx tsx packages/zeroad-adapter/src/arena/run-arena-loop.ts
```

### 3. Watch the Magic

During matches:
- AI models make strategic decisions
- Trash talk is generated and captured
- Voice is synthesized to WAV files
- Audio files created in `.data/audio/trash_talk/`
- ELO ratings updated
- Results logged to console

---

## What Happens During a Match

1. **Setup** (5-10 sec)
   - Kill any running 0 A.D.
   - Start fresh game with RL Interface
   - Wait for game to load

2. **Gameplay** (2-4 min)
   - Ollama AI makes decisions every tick
   - Game advances turn-by-turn
   - Trash talk generated randomly
   - Voice synthesized in real-time

3. **Completion** (10-20 sec)
   - Match ends (winner determined)
   - Results saved
   - ELO ratings updated
   - Game closed

4. **Next Match** (5-10 sec)
   - Next map/civs selected
   - Process repeats

## Voice Synthesis

### How It Works

1. Trash talk text generated by AI
2. Piper TTS synthesizes to voice (50-200ms)
3. Audio saved to `.data/audio/trash_talk/trash_talk_TIMESTAMP.wav`
4. Can be played via media player or integrated into streaming

### Audio Files

- **Size:** 30-100 KB per message
- **Format:** WAV (PCM 16-bit, 22050 Hz)
- **Quality:** Natural and clear
- **Location:** `.data/audio/trash_talk/`

### Playback

```bash
# Windows (PowerShell)
Get-ChildItem .data/audio/trash_talk/*.wav | ForEach-Object { & $_ }

# macOS
open .data/audio/trash_talk/*.wav

# Linux
vlc .data/audio/trash_talk/

# Or any media player
```

## Configuration Options

### AI Models

```bash
# Fast (tinyllama, 637 MB)
OLLAMA_MODEL=tinyllama:latest npx tsx packages/zeroad-adapter/src/arena/run-arena-loop.ts

# Balanced (mistral, 4.1 GB)
OLLAMA_MODEL=mistral:latest npx tsx packages/zeroad-adapter/src/arena/run-arena-loop.ts

# High Quality (neural-chat, 4.1 GB)
OLLAMA_MODEL=neural-chat:latest npx tsx packages/zeroad-adapter/src/arena/run-arena-loop.ts
```

### Decision Frequency

```bash
# Every tick (default, most responsive)
npx tsx packages/zeroad-adapter/src/arena/run-arena-loop.ts --freq 1

# Every 5 ticks (5x faster)
npx tsx packages/zeroad-adapter/src/arena/run-arena-loop.ts --freq 5

# Every 10 ticks (10x faster)
npx tsx packages/zeroad-adapter/src/arena/run-arena-loop.ts --freq 10
```

## Troubleshooting

### Issue: "Ollama not running"
```bash
# Start Ollama in new terminal
ollama serve
```

### Issue: "Model not available"
```bash
# Check available models
ollama list

# Download if missing
ollama pull tinyllama
ollama pull mistral
```

### Issue: Audio files not created
Check logs for TTS errors. Voice model files must be in:
```
packages/zeroad-adapter/voices/
```

### Issue: Build fails
```bash
npm run build
```
