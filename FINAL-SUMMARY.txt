═════════════════════════════════════════════════════════════════════════════
ARCHITECTURE
═════════════════════════════════════════════════════════════════════════════

Data Flow:
  Ollama (AI) → Chess Arena (game loop) → WebSocket (broadcast)
               ↓
          Real Chess Game (move execution)
               ↓
          RealChessGame.executeMove() → wsServer.emitMovePlayed()
               ↓
          Browser WebSocket → React Hook → Component State → Render

Per Move Latency: <50ms from execution to browser display
